3. Usage

3.1 General Usage

KYahoo is designed to work like the Official Yahoo Messenger. Information on that and creating pager accounts is available at Yahoo's site.

3.2 Menus

File

  • Options

    This is the configuration dialog. You can enter your login and password with this option. The configuration is only available when you are not logged into the pager.

  • Save Options

    This option will be highlighted when changes have been made after closing the configuration dialog. If you don't save options right away they will be automatically done when you close KYahoo.

  • Quit

    This logs out of Yahoo and closes KYahoo.

Yahoo

  • Logon

    This will log into Yahoo. Make sure you have entered your login and password already in the configuration dialog.

  • Logoff

    This will logoff of Yahoo. It does not close the window though.

  • New Conference

    This will start a new conference. You have to select users to invite to the conference. The dialog gives you a list of online buddies and also allows you to type in additional buddies.

  • Send offline message

    This will send an offline message to someone in your buddylist who isn't logged in.

  • Add User

    This dialog will add a new user to your buddylist.

  • Remove User

    This dialog will remove a user from your buddylist.

  • Debug Messages

    This dialog will display debug messages.

Help

  • Contents

    This will display the help browser.

  • Report Bug

    This dialog allows you to submit a bug. I prefer if you use the sourceforge page instead, but its here for convience.

  • About KYahoo

    This dialog displays information about the current version of KYahoo.

  • About KDE

    This dialog displays information about the current version of KDE.
