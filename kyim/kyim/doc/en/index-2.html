2. Installation

2.1 How to obtain KYahoo

KYahoo is available for download from the KYahoo web page, kyahoo.sourceforge.net.

2.2 Requirements

KYahoo requires KDE version 2.x and Qt version 2.x.

2.3 Compilation and installation

In order to compile and install KYahoo on your system, type the following in the base directory of the KYahoo distribution:

% ./configure
% make
% make install

Since KYahoo uses 

autoconf
you should have not trouble compiling it. Should you run into problems please report them to the the author at Jay Lubomirski
