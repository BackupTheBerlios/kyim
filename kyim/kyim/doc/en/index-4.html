4. Configuration Options

Login

  • Yahoo ID

    This is your Yahoo login name.

  • Password

    Your Yahoo password.

  • Login under Invisible Mode

    This option allows you to log in and not look like you are logged in to anyone else.

  • Login on startup

    This option will automatically log into Yahoo when you start KYahoo.

Misc

  • Accept Message from friends only?

    This feature isn't implemented yet. It will be tied into the ignore list of the real Yahoo pager and will only allow messages from people in your buddy list.

  • Show online friends only?

    This feature will only show buddies that are online in the buddylist.

  • Dynamically resize buddylist?

    This feature will attempt to resize the main KYahoo window to fit the custom status messages in the window.

  • Use graphical emoticons in chat window?

    This option will use graphical emoticons instead of the text version.

  • Allow Idle state

    This feature will set your status from Available to Idle after a certain amount of time. If your status is not Available, you will not transistion to Idle state even if you are logged in.

LibYahoo

  • Connection Mode

    This feature will select your connection mode. Normal mode is the default connection method and should be used if you can be directly connected to the internet. HTTP mode will attempt to only use the HTTP protocol to talk to Yahoo. HTTP-Proxy mode will use a HTTP proxy to talk to Yahoo.

  • Proxy Host

    Use this option with the HTTP-Proxy connection mode to specify the hostname of the HTTP proxy.

  • Proxy Port

    Use this option with the HTTP-Proxy connection mode to specify the port number of the HTTP proxy host.

Sound

  • Play sound for Buddy Logon

    This option will play the selected .wav file when someone on your buddylist logs into Yahoo.

  • Play sound for Buddy Logoff

    This option will play the selected .wav file when someone on your buddylist leaves Yahoo.

  • Play sound for new message

    This option will play the selected .wav file when a new message has arrived. It will only play if there is no chat window open for this message or if the chat window that is open isn't currently the active window.

  • Play a buzz sound

    This option will play the selected .wav file when a user buzz's you, or you type < ding > in chat window.
